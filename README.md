## 项目运行初始化运行方式
```bash
npm i

npm run dev
```


## 项目说明
整体项目基于el-tabs结构，每个tab页都是一个组件，组件之间通过tabs组件功能进行切换
!(tabs参考)[https://element-plus.org/zh-CN/component/tabs.html]
